# HYDRO-EQUITY-ENGINE_SMC
Hydro-Equity Engine — A hydraulic governance platform for Solapur Municipal Corporation. Parses real SMC GIS pipeline data, runs WNTR hydraulic simulation on the live pipe network, computes zone-wise pressure equity scores (HEI), detects leaks via multi-signal CLPS, and visualises everything on a Leaflet dashboard. Built for SAMVED-2026.
